## 💡 Идея

Все сайты с судоку одинаковые — сетка, цифры, кнопка «проверить». Они не объясняют логику, не учат думать, не удерживают пользователя.

**СудокуРазум** решает это: ИИ-тренер анализирует каждую ячейку в реальном времени и объясняет *почему* то или иное число подходит. Это превращает обычную игру в обучающий инструмент.

## ✨ Что сделано

### Уровень «Сильный» ✅
- 🎲 **Генератор уникальных судоку** — алгоритм backtracking с проверкой единственности решения, каждая игра уникальна
- 🎚️ **4 уровня сложности** — Лёгкий (35 подсказок) → Эксперт (17 подсказок)
- ⚡ **Проверка правил в реальном времени** — конфликты подсвечиваются красным мгновенно
- 📝 **Режим заметок** — помечай кандидатов в ячейках как профессиональный игрок
- 💡 **Система подсказок** — 3 подсказки на игру, не даёт читерить бесконечно
- ⏱️ **Таймер + счётчик ошибок** — честная статистика каждой сессии
- ↩️ **История отмен** — исправляй ошибки без штрафа
- 🌙 **Тёмная / светлая тема** — комфортная игра в любое время
- 📊 **Статистика сессии** — сыграно, побед, лучшее время, процент побед
- 📱 **Адаптивный дизайн** — удобно играть с телефона

### Уровень «Великий» ✨
- 🤖 **ИИ-тренер** — анализирует выбранную ячейку, объясняет возможные числа и логику метода исключения. Настоящая педагогическая ценность
- 💰 **Pro-монетизация** — полноценное модальное окно с двумя тарифами, описанием фич и реальным сбором email через Stripe-ready форму
- 📧 **Waitlist через Formspree** — каждый кто оставляет email попадает в базу, данные приходят на почту основателя
- 📤 **Share результата** — вирусный механик через Web Share API

## 🛠️ Технологии

| Стек | Почему |
|------|--------|
| **Vanilla JS** | Без фреймворков — максимальная скорость загрузки |
| **Inline styles** | Гарантированная работа тем в любом окружении |
| **Formspree** | Сбор email без бэкенда |
| **Netlify** | Бесплатный хостинг с CDN |
| **Web Share API** | Нативный шеринг результатов |

**Алгоритм генерации:**
1. Backtracking с рандомизацией → заполняет полную сетку
2. `countSolutions()` → проверяет единственность решения
3. Удаляет ячейки пока не достигнет нужного количества подсказок
